DevOps Workflow

Separate infrastructure, platform, SDK, and application delivery.

Flow

  • Terraform provisions Azure infrastructure and the portfolio Static Web App.
  • Kubernetes platform manifests create tenant isolation and NAP spot node provisioning policies.
  • The platform SDK is built and published as a package artifact.
  • GitHub Actions builds boutique microservice images in parallel and pushes them to ACR.
  • GitHub Actions packages the reusable Helm chart and pushes it to ACR OCI.
  • Flux reconciles the OCI chart, app HelmReleases, and app settings ConfigMaps.
