Observability

Application Insights, Log Analytics, and ELK examples for platform operations.

Application Insights KQL

requests
| summarize count(), avg(duration) by cloud_RoleName, resultCode
| order by count_ desc

Log Analytics KQL

ContainerLogV2
| where TimeGenerated > ago(1h)
| summarize count() by Namespace, PodName, LogLevel

ELK Dashboard Ideas

  • Tenant request rate and error rate.
  • Gateway 4xx and 5xx by hostname.
  • Container restart and eviction trend.
  • Spot node interruption and pending pod trend.
  • Application log correlation by trace ID.
