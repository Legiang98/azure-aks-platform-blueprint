Platform SDK

One application-facing package for common platform integrations.

The SDK wraps Managed Identity, Key Vault secret access, and Application Insights setup behind a small developer interface.

Usage

import { createPlatformClient } from "@aks-platform/sdk";

const platform = await createPlatformClient();
await platform.initializeTelemetry();

const value = await platform.keyVault?.getRequiredSecret("app-config");

Artifact Boundary

Publish the SDK to a private npm registry such as GitHub Packages or Azure Artifacts. Do not commit registry tokens or environment-specific cloud identifiers.
