Platform Shape

The platform uses a shared VPN VNet with regional AKS and data VNets, private endpoint resolution through Private DNS, a shared Gateway API entry point, Kubernetes namespace-level tenant isolation, and split on-demand/spot capacity.

Developer Interface

Applications consume platform capabilities through a small SDK package instead of repeating Key Vault, Managed Identity, and Application Insights wiring in every service.
